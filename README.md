# rubik
Agent for LLM with enterprise using

## File directory structure

|-- api        // 对外公开API

|-- builder    // 构建的脚本等

|-- cli        // 命令行工具

|-- cmd        // 可执行程序，包含main

|-- contrib    // 一些有用的脚本或文件，非核心项目文件

|-- docks      // 项目文档

|-- internal   // 项目中使用的包

|-- pkg        // 公共使用的包

|-- plugin     // 插件
